# Traductor con Voz Multilingüe

Esta es una aplicación web ligera desarrollada en HTML, CSS y JavaScript, diseñada para traducir texto entre múltiples idiomas y reproducir automáticamente el resultado mediante síntesis de voz en el idioma seleccionado.

El proyecto funciona completamente en el navegador y es compatible con GitHub Pages, sin necesidad de servidores ni configuraciones adicionales. La traducción se realiza de forma automática y la lectura por voz utiliza las capacidades nativas del navegador, ofreciendo una experiencia rápida, accesible y multiplataforma.

Características principales

Traducción automática de texto entre múltiples idiomas

Reproducción por voz del texto traducido

Selección manual del idioma de salida

Funcionamiento 100% del lado del cliente

Compatible con navegadores modernos (especialmente Google Chrome)

Ideal para aprendizaje de idiomas, accesibilidad y demostraciones técnicas

Tecnologías utilizadas

HTML5

CSS3

JavaScript

Web Speech API (Text-to-Speech)

Servicio de traducción automática de Google (endpoint público)

Uso recomendado

Este proyecto es adecuado para fines educativos, demostrativos o personales, como práctica de desarrollo
